# Shopventory 

El objetivo es crear una app que facilite a los pequeños negocios la gestión de su inventario. La aplicación permitirá a los empleados registrar productos, actualizar cantidades disponibles, recibir alertas cuando el stock sea bajo y visualizar datos relacionados con los pedidos al almacén.

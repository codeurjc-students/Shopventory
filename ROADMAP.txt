Funcionalidades implementadas para probar
Autenticación
 Login con admin ( admin@shopventory.com / Admin1234! )
 Login con usuario estándar ( user@shopventory.com / User1234! )
 Intentar login con cuenta pendiente ( pending@shopventory.com / Pending1234! ) → mensaje de error
 Logout
Dashboard
 Ver KPIs: total productos, pedidos, valor de inventario, alertas de stock
Productos
 Ver listado paginado de productos
 Buscar producto por nombre/SKU
 Crear nuevo producto (solo admin) → botón "New Product"
 Ver detalle de un producto (icono ojo)
 Editar producto (solo admin)
 Actualizar stock desde el detalle (cantidad positiva = entrada, negativa = salida)
 Eliminar producto (solo admin)
 Ver sección "Low Stock" en el dashboard (producto USB-C Hub tiene stock 3, umbral 5)
Pedidos
 Ver listado de pedidos
 Crear pedido de tipo SALE con líneas de producto
 Crear pedido de tipo PURCHASE seleccionando proveedor
Proveedores
 Ver listado de proveedores
 Crear proveedor (solo admin)
 Editar proveedor (solo admin)
 Eliminar proveedor (solo admin)
Empleados (solo admin)
 Ver listado de empleados
 Crear empleado
 Editar empleado
 Eliminar empleado
Gestión de usuarios (solo admin)
 Ver listado de usuarios
 Aprobar al usuario registrado en el paso de autenticación
 Verificar que el usuario aprobado puede hacer login
 Deshabilitar/habilitar una cuenta
Control de acceso
 Iniciar sesión como usuario estándar (user@shopventory.com)
 Verificar que NO ve el menú de Empleados ni Usuarios
 Verificar que NO puede acceder a /employees directamente (redirige a 403)
 Verificar que NO puede crear/editar/eliminar productos
Páginas de error
 Acceder a una ruta inexistente → página 404
 Acceder a /403 directamente → página 403
